İlk olarak şunu söylemek isterimki herkes gibi bende aptalca programlama yapmaya karşıyım mesela az sonra yapacağımız yıldız ile baklava şekli çıkarmak gibi ,fakat bazı arkadaşlarımın katıldığı yerler de yok değil söylediklerine göre böyle yaparak döngüleri daha iyi öğrenecekmişim herneyse bakalım nasıl yapmışım.

public class Baklava {
     public static void main(String[] args) {
         int a,b,c=22;
         final int satir=1;
         for (int i=10;i>=satir;i--)
         {
            c-=2;
             for(a=1;a<=i;a++)
             System.out.print("*");
        for(b=20;b>=c;b-=1)
             System.out.print(" ");
        for(a=1;a<=i;a++)
             System.out.print("*");
             System.out.println();
             }
        final int satir2=10;
           int d=21;
             for (int j=1;j<=satir2;j++)
         {
            d-=2;
             for(a=1;a<=j;a++)
             System.out.print("*");
        for(b=1;b<=d;b+=1)
             System.out.print(" ");
        for(a=1;a<=j;a++)
             System.out.print("*");
             System.out.println();
         }  
         }}

Görüldüğü gibi biraz uğraşınca oluyor tabiki şimdi birsürü kişi çıkacak şöyle de olabilirdi diyecek ve bende bunu bekleyeceğim:)
